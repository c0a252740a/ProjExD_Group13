# 横シューティングゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 自機を操作して「階層」を登っていく。毎階層ごとにスキルが選べる。最終階層までいくとエンディングが見れる。
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 矢印キーで自機を操作し，スペースキー押下による弾丸で敵を撃つ
* 敵に当たるとゲームオーバーとなる
* 操作キー WASD , SPACE発射 , Q/E/Rスキル

## ゲームの実装
### 共通基本機能
* 背景画像
* 自機の移動 ok
* 弾丸の発射(味方/敵) ok
* 敵(弾を撃つ) ok
* アイテムオブジェクト
* スキル選択
* 強制スクロール
* ステージの端に着いたら、スキル選んで次の階層へ
* 楽しい地形(当たり判定にあるスプライト)
* スコア ok
* HPはデフォルト3 ok

### 分担追加機能
#### 全員
1. 一人一つのボス
2. 一人一つのステージ(BGM)
3. 一人一つのスキル
